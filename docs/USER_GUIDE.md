# How to Use This App

**Simple step-by-step instructions for editing Facebook Marketplace listings.**

---

## STEP 1: Upload Your File

### What You Need
- An Excel file (.xlsx or .xls) with your product listings
- OR a CSV file (.csv)

### What To Do

1. **Look for the big upload area** in the middle of the screen
   - It says "Drag and drop Excel files here"
   - It has a cloud upload icon

2. **Choose ONE of these options:**

   **Option A: Drag and Drop**
   - Open your file folder
   - Find your Excel file
   - Click and hold the file
   - Drag it onto the upload area
   - Let go

   **Option B: Click to Browse**
   - Click anywhere in the upload area
   - A file picker window opens
   - Find your Excel file
   - Click "Open"

### What Happens Next
- The file uploads (takes 1-2 seconds)
- Your data appears in a table below
- You'll see columns: TITLE, PRICE, CONDITION, DESCRIPTION, CATEGORY, OFFER SHIPPING

### If Upload Fails
- Make sure your file has these column names: TITLE, PRICE, CONDITION, DESCRIPTION, CATEGORY
- File must be .xlsx, .xls, or .csv format

## STEP 2: Edit a Cell

### What To Do

1. **Find the cell you want to change**
   - Look in the table
   - Example: You want to change a price

2. **Click directly on the cell**
   - The cell turns into a text box
   - You can now type

3. **Type your new value**
   - Example: Change "100" to "150"

4. **Save your change**
   - Press **Enter** on your keyboard
   - OR click outside the cell

### What Happens Next
- The cell saves automatically
- You'll see "Saved at [time]" appear briefly at the top
- The change is now in your data

### If You Made a Mistake
- Press **Escape** before pressing Enter to cancel
- OR use the Undo button (see Step 6)

---

## STEP 3: Add a New Row

### What To Do

1. **Find any existing row** in the table

2. **Look for the Actions column** (last column on the right)

3. **Click the copy icon** (looks like two overlapping squares)

### What Happens Next
- A new row appears below the one you clicked
- It's an exact copy of that row
- You can now edit the new row (see Step 2)

---

## STEP 4: Delete a Row

### What To Do

1. **Find the row you want to delete**

2. **Look for the Actions column** (last column on the right)

3. **Click the trash icon** (looks like a garbage can)

### What Happens Next
- The row disappears immediately
- **Warning:** You can't get it back unless you use Undo (see Step 6)

---

## STEP 5: Search for Something

### What To Do

1. **Look at the top of the table**
   - You'll see a search box that says "Search listings..."

2. **Click in the search box**

3. **Type what you're looking for**
   - Example: Type "solar" to find all solar panels
   - Example: Type "150" to find all items priced at $150

### What Happens Next
- The table shows only rows that match your search
- Other rows are hidden (not deleted)

### To See All Rows Again
- Delete the text in the search box
- OR click the X button in the search box

---

## STEP 6: Undo a Mistake

### What To Do

1. **Look at the top of the screen**
   - Find the Undo button (curved arrow pointing left)

2. **Click the Undo button**

### What Happens Next
- Your last change is reversed
- Example: If you deleted a row, it comes back
- Example: If you edited a cell, it goes back to the old value

### To Redo
- Click the Redo button (curved arrow pointing right)
- This brings back the change you just undid

---

## STEP 7: Export Your File

### What To Do

1. **Look at the top-right of the screen**
   - Find the green "Export for FB" button

2. **Click "Export for FB"**

3. **A preview window opens**
   - Shows what your file will look like
   - Check that everything looks correct

4. **Click "Download Excel File"** (green button at bottom)

### What Happens Next
- A file downloads to your computer
- File name: `Facebook_Marketplace_Bulk_Upload_[date].xlsx`
- This file is ready to upload to Facebook Marketplace

### Where Is My File?
- Check your Downloads folder
- Look for a file starting with "Facebook_Marketplace_Bulk_Upload"

## Common Questions

### "Where did my file go after I uploaded it?"
- It's in the table below the upload area
- Scroll down to see it

### "Can I upload multiple files?"
- Yes! Upload one file, then upload another
- They combine into one big table

### "How do I save my work?"
- The app saves automatically every time you edit a cell
- To save to your computer, use Export (Step 7)

### "I accidentally deleted a row!"
- Click the Undo button immediately (Step 6)

### "The Export button is gray and won't click"
- You need to upload a file first (Step 1)

### "I want to change the price of 50 items at once"
- Use Bulk Edit (Step 8)
